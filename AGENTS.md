This repository is for a feedreader. There is a fully working version written in rust at the root, but we are converting it to an Elixir app in the feedreader folder.

Read feedreader/SPEC.md to understand the elixir changes. Use an explore agent if you need to search the rust version for examples.

The new elixir project uses mise for erlang/elixir deps and developer tasks. Read feedreader/mise.toml for more information
